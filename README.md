# README

This is a SaaS Boilerplate for Fast MVP launch

Tech stacks:
- Ruby +3
- Rails v8
- Turbo v8
- Stimulus JS
- Tailwindcss v4 & DaisyUI
- Importmap
- Devise for auth
- Erb lint
- Rubocop mixed of Omakase & Standard RB
- StandardJs
- Postgresql for database
- Sqlite for Solid trifecta
- Kamal for deployment
